# JAIST 丹研究室 学位論文・研究トレンド総覧
JAIST (北陸先端科学技術大学院大学) 丹研究室（情報科学）の修士・博士論文データベースおよび研究トレンドの可視化ダッシュボードです。

## 🌟 主な機能
1. **概要統計 (Stats Panel)**: 学位論文の総数、修士・博士それぞれの本数、および固有キーワード数を一目で把握できます。
2. **年度別学位論文発表数推移 (Stacked Bar Chart)**: 修士・博士の発表本数の歴史的推移を積み上げ棒グラフで可視化します。
3. **キーワード出現頻度ランキング (Keyword Ranking)**: 頻出する主要な研究キーワードを横棒グラフで表示します。「全期間」「近15年」「近10年」「近5年」の時間スパンで切り替え可能です。
4. **主要研究テーマの時系列推移ヒートマップ (Timeline Heatmap)**: 上位15個のキーワードが各年度で何回出現したかを強弱（色の濃淡）でグリッド表示し、研究テーマの流行や技術変遷の歴史を追うことができます。
5. **論文データベース検索 (Interactive Search)**: Title, Title (EN), Author, Author Ruby, Keywords, Supervisor を対象としたリアルタイムのあいまい検索および学位・年度・注目キーワードでの絞り込みが可能です。
   * **カード表示モード**: 現代的で詳細な情報をグリッド上に配置して閲覧できます。
   * **年度別一覧モード (Classic Gray Style)**: JAISTの伝統的なグレースタイルのテーブル表示を忠実に再現した表示モードです。
6. **管理者ポータル (admin.html)**: 簡易的なデータ登録（LocalStorageへの一時登録）や、DSpaceからのHTMLペーストによる一括パース、更新された `data.js` 用JSONデータの書き出し機能を提供します。

## 🛠️ 技術スタック
* **Frontend**: HTML5, Vanilla JavaScript, CSS3 (カスタムデザイン)
* **Library**: Chart.js (グラフ描画), FontAwesome (アイコンフォント), Google Fonts (Noto Sans JP, Outfit)
* **Data Source**: [JAIST学術研究成果リポジトリ (DSpace)](https://dspace.jaist.ac.jp/dspace/) より取得

## 🚀 使い方
本プロジェクトは静的なWebサイトとして構築されているため、サーバーのセットアップは不要です。

1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` を任意のブラウザでダブルクリック、または Live Server 等のローカル開発用サーバーで開くだけで起動します。

## 📂 ディレクトリ構成
* `index.html` - メインのダッシュボード画面
* `admin.html` - 管理者用データインポート・出力ツール
* `data.js` - 修士・博士論文の静的データベースファイル (JSON構造)
* `css/`
  * `style.css` - メイン画面のスタイル定義
  * `admin.css` - 管理者画面のスタイル定義
* `js/`
  * `app.js` - UI操作、検索、フィルタリング、初期化ロジック
  * `charts.js` - 棒グラフ、キーワードランキング、ヒートマップの描画ロジック
  * `admin.js` - 管理者機能用（データのパース、インポート/エクスポート等）のロジック
