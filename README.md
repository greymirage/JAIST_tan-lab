# JAIST 丹研究室 学位論文・研究トレンド総覧
JAIST (北陸先端科学技術大学院大学) 丹研究室（情報科学）の修士・博士論文データベース、研究トレンドの可視化ダッシュボード、および対話型AIアシスタントツールです。

## 🌟 主な機能
1. **概要統計 (Stats Panel)**: 学位論文の総数、修士・博士それぞれの本数、および固有キーワード数を一目で把握できます。
2. **年度別学位論文発表数推移 (Stacked Bar Chart)**: 修士・博士の発表本数の歴史的推移を積み上げ棒グラフで可視化します。
3. **キーワード出現頻度ランキング (Keyword Ranking)**: 頻出する主要な研究キーワードを横棒グラフで表示します。「全期間」「近15年」「近10年」「近5年」の時間スパンで切り替え可能で、表示範囲はデータセットに応じて自動計算されます。
4. **主要研究テーマの時系列推移ヒートマップ (Timeline Heatmap)**: 上位15個のキーワードが各年度で何回出現したかを色の濃淡でグリッド表示し、研究トレンドの変遷を追うことができます。
5. **論文データベース検索 (Interactive Search)**: タイトル、英語タイトル、著者、著者（ヨミ）、キーワード、指導教員を対象としたリアルタイムのあいまい検索および絞り込みが可能です。
   * **カード表示モード**: 現代的で詳細な情報をグリッド上に配置して閲覧できます。
   * **年度別一覧モード (Classic Gray Style)**: JAISTの伝統的なグレースタイルのテーブル表示を忠実に再現した表示モードです。
6. **AIアシスタント (ai.html)**: 過去の論文データをもとに、Gemini 3.1 Flash-Lite AIが論文の対話的検索や研究テーマ発掘の相談、アイデアの壁打ちに応じます（RenderにデプロイされたセキュアなAPI中継プロキシ経由で安全に動作します）。対話履歴はタイムスタンプ付きの月別テキストファイルとしてダウンロード保存可能です。
7. **管理者ポータル (admin.html)**: 簡易的なデータ登録（LocalStorageへの一時登録）や、DSpaceからのHTMLペーストによる一括パース、更新された `data.js` 用JSONデータの書き出し機能を提供します。

## 🛠️ 技術スタック
* **Frontend**: HTML5, Vanilla JavaScript, CSS3 (カスタムデザイン)
* **Libraries**: Chart.js (グラフ描画), FontAwesome (アイコンフォント), Google Fonts (Noto Sans JP, Outfit)
* **LLM API**: Google Gemini 3.1 Flash-Lite (Renderプロキシ中継経由)
* **Data Source**: [JAIST学術研究成果リポジトリ (DSpace)](https://dspace.jaist.ac.jp/dspace/) より取得

## 🚀 使い方
本プロジェクトは静的なWebサイトとして構築されているため、サーバーのセットアップは不要です。

1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` を任意のブラウザでダブルクリック、または Live Server 等のローカル開発用サーバーで開くことで起動します。

## 📂 ディレクトリ構成
```
JAIST_tan-lab/
├── index.html          # メインのダッシュボード画面
├── ai.html             # AIアシスタントチャット画面
├── admin.html          # 管理者用データインポート・出力ツール
├── .gitignore          # Gitコミット除外ルール
├── data/
│   ├── data.js         # 統合論文データベース (JSON構造)
│   ├── data_ai.js      # AIプロンプト用の軽量圧縮テキストデータベース
│   └── raw/            # 元のテキストデータ保管用フォルダ
├── css/
│   ├── style.css       # メイン画面および基本カードレイアウトのスタイル定義
│   ├── ai.css          # AIチャット画面専用のフラットスタイル定義
│   └── admin.css       # 管理者画面のスタイル定義
└── js/
    ├── normalize.js    # 共通キーワード正規化マップおよびクリーン処理関数
    ├── app.js          # UI操作、検索、フィルタリング、初期化ロジック
    ├── charts.js       # 棒グラフ、キーワードランキング、ヒートマップの描画ロジック
    ├── ai.js           # Gemini API連携、対話ログ保存ダウンロード処理ロジック
    └── admin.js        # 管理者用データのパース、インポート/エクスポートロジック
```
